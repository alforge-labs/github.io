    CLI Reference¶

    A complete catalog of every command group provided by the forge CLI. Detailed parameters and output examples for each group are documented on the per-group pages linked below.

    Scaffolded pages

    Detailed references (parameters, output examples, error codes) for each group are being filled in incrementally via per-group issues. The "Subcommands" tables on this page are kept in sync with the implementation.

    Core Command Groups¶

    The commands you'll use most often in real strategy development. Each has a dedicated page.

    Group Description Details
    backtest Run backtests and analyze results backtest →
    optimize Parameter optimization (Bayesian, grid, walk-forward) optimize →
    strategy Create, register, and manage strategy JSON strategy →
    data Fetch and update historical data data →
    journal Track run history, tags, and verdicts journal →
    live Live trading analysis and records live →

    Other Commands¶

    Authentication, utilities, and supporting features are bundled on the Other commands page.

    Group Description Link
    auth Whop OAuth login / logout and authentication status other#auth →
    init Initial project setup other#init →
    pine Convert between strategy JSON and TradingView Pine Script other#pine →
    indicator List and inspect supported technical indicators other#indicator →
    idea Manage and search investment ideas other#idea →
    altdata Fetch and manage alternative data (sentiment, etc.) other#altdata →
    pairs Pair trading (cointegration tests) other#pairs →
    docs Browse bundled documentation other#docs →

    Built-in Help¶

    Every command supports --help.

    forge --help                         # Top-level command list
forge backtest --help                # Subcommands of the backtest group
forge backtest run --help            # Detailed parameters for a specific subcommand
