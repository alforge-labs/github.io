forge backtest run

指定した戦略とシンボルでバックテストを実行します。結果は SQLite に保存され、JSON 出力も可能です。

forge backtest run <SYMBOL> --strategy <STRATEGY_NAME> [OPTIONS]

使用例

forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1
forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1 --json
forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1 --start 2021-01-01 --end 2024-12-31

forge optimize run

Optuna を使ったベイズ最適化でパラメータ空間を効率的に探索します。

forge optimize run <SYMBOL> --strategy <STRATEGY_NAME> --trials 200 -j 4

forge backtest wft

ウォークフォワードテストを実行します。過学習を検出するための標準的な検証手法です。

forge backtest wft CL=F --strategy cl_hmm_bb_rsi_v1 --folds 5

forge license activate

LemonSqueezy のライセンスキーを認証します。認証情報は ~/.forge/license.json にキャッシュされます。

forge license activate <LICENSE_KEY>
コマンド説明
forge license status現在のライセンス状態を表示
forge license deactivateこのデバイスのライセンスを無効化

forge data update

設定ファイルに記載されたシンボルのヒストリカルデータを最新日まで差分更新します。

forge data update
forge data update --symbol CL=F
forge data update --full

forge backtest run

Runs a backtest for a symbol and strategy. Results are stored in SQLite and can also be printed as JSON.

forge backtest run <SYMBOL> --strategy <STRATEGY_NAME> [OPTIONS]

Examples

forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1
forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1 --json
forge backtest run CL=F --strategy cl_hmm_bb_rsi_v1 --start 2021-01-01 --end 2024-12-31

forge optimize run

Uses Optuna to search the strategy parameter space efficiently.

forge optimize run <SYMBOL> --strategy <STRATEGY_NAME> --trials 200 -j 4

forge backtest wft

Runs a walk-forward test to validate robustness and reduce overfitting risk.

forge backtest wft CL=F --strategy cl_hmm_bb_rsi_v1 --folds 5

forge license activate

Activates a LemonSqueezy license key. Activation data is cached in ~/.forge/license.json.

forge license activate <LICENSE_KEY>
CommandDescription
forge license statusShow the current license status
forge license deactivateDeactivate this device

forge data update

Updates historical data for symbols defined in the configuration file.

forge data update
forge data update --symbol CL=F
forge data update --full
