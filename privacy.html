プライバシーポリシー

最終更新日: 2026年4月25日

Alforge Labs（以下「当社」）は、AlphaForge CLI の提供にあたり収集する個人情報の取り扱いについて、以下の通りプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 収集する情報

1.1 購入時に収集する情報

LemonSqueezy を通じたライセンス購入時に、以下の情報が収集されます。これらの情報は LemonSqueezy が直接収集・管理します。

  • 氏名（または会社名）
  • メールアドレス
  • 決済情報（クレジットカード情報等 — LemonSqueezy の決済システムで処理され、当社は保持しません）
  • 国・地域情報（税務目的）

1.2 ライセンス認証時に収集する情報

forge license activate 実行時に、以下の情報が LemonSqueezy の License API に送信されます。

  • ライセンスキー
  • インスタンス名（デフォルト: マシンのホスト名）
  • アクティベーション日時

1.3 収集しない情報

AlphaForge CLI はローカル完結型ソフトウェアです。以下の情報は当社のサーバーには一切送信されません。

  • バックテストの設定・結果
  • 最適化パラメータ・戦略データ
  • 取引履歴・ポジション情報
  • ローカルのファイルシステム情報
  • 使用状況・操作ログ（テレメトリ無効）

2. 情報の利用目的

収集した情報は以下の目的のみに使用します。

  • ライセンスキーの発行・認証・管理
  • 購入確認メール・領収書の送付
  • 製品アップデートのお知らせ（登録ユーザーのみ、配信停止可）
  • サポート対応
  • 法令上の義務履行（税務・会計処理）

3. 第三者への情報提供

当社は以下の場合を除き、個人情報を第三者に提供しません。

提供先提供する情報目的
LemonSqueezy 購入情報・ライセンスキー 決済処理・ライセンス管理
法執行機関等 必要最小限の情報 法令上の要請がある場合のみ

マーケティング目的での第三者へのデータ販売・共有は行いません。

4. データの保存期間

  • 購入情報・ライセンス情報: LemonSqueezy のデータ保持ポリシーに準拠（通常は取引完了後7年間）
  • サポートメール: 対応完了後2年間
  • ローカルの認証キャッシュ (~/.forge/license.json): ユーザーが手動で削除するまで、または forge license deactivate を実行するまで保持

5. セキュリティ

当社は個人情報を保護するために適切な技術的・組織的措置を講じています。ただし、インターネット経由のデータ送信は完全に安全であることを保証できません。

6. ユーザーの権利

以下の権利を行使する場合は、下記連絡先までお問い合わせください。

  • アクセス権: 保有する個人情報の開示を求める権利
  • 訂正権: 不正確な個人情報の訂正を求める権利
  • 削除権: 個人情報の削除を求める権利（法令上の保存義務がある場合を除く）
  • 処理制限権: 個人情報の処理制限を求める権利

削除要求を受け付けた場合、当社が保持する情報と、LemonSqueezy に対する削除要求の代行を30日以内に行います。

7. Cookie・トラッキング

本ウェブサイト（alforge-labs.github.io）では Google Analytics を使用してアクセス状況を分析しています。収集されるデータは匿名化されており、個人を特定する情報は含まれません。ブラウザの設定から Cookie を無効化することで計測を拒否できます。

8. 適用範囲

本ポリシーは Alforge Labs が運営する以下に適用されます。

  • ウェブサイト: alforge-labs.github.io
  • AlphaForge CLI ソフトウェア

9. ポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合は登録メールアドレスへの通知またはウェブサイトへの掲載によりお知らせします。

10. お問い合わせ

個人情報の取り扱いに関するお問い合わせ・開示・削除要求は以下にご連絡ください。

Email: support@alforge-labs.com
