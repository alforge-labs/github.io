クイックインストール

ターミナルで実行します。インストーラーが最新バイナリをダウンロードし、/usr/local/bin に配置します。

curl -sSL https://alforge-labs.github.io/install.sh | bash

PowerShell（管理者権限不要）で実行します。バイナリを %USERPROFILE%\.forge\bin にインストールし、PATH を自動設定します。

irm https://alforge-labs.github.io/install.ps1 | iex

ライセンス認証

購入完了メールに記載されているライセンスキーで認証します。認証情報は ~/.forge/license.json に保存されます。

forge license activate <YOUR_LICENSE_KEY>

主要コマンド

forge backtest run
単一銘柄に戦略を適用してバックテストを実行する。
forge backtest run SYMBOL --strategy ID
--strategy--start--json
cli-reference で詳細 →
forge optimize run
Optuna によるパラメータ最適化（TPE）。多目的最適化にも対応。
forge optimize run SYMBOL --strategy ID
--metric--trials--apply
cli-reference で詳細 →
forge optimize walk-forward
時系列をウィンドウ分割し IS 最適化 → OOS 評価で過学習耐性を測定。
forge optimize walk-forward SYMBOL --strategy ID --windows N
--windows--metric
cli-reference で詳細 →
forge data fetch
OHLCV をプロバイダーから取得し Parquet で保存。
forge data fetch SYMBOL --period 5y --interval 1d
--period--interval--watchlist
cli-reference で詳細 →
forge strategy save
戦略 JSON をレジストリに登録（任意で Journal に記録）。
forge strategy save FILE_PATH
--force
cli-reference で詳細 →
forge journal show
戦略の全履歴（スナップショット・Runs・タグ・ライブサマリ）を表示。
forge journal show STRATEGY_ID
