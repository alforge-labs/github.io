3 ステップで AlphaForge を動かす

Whop 登録 → インストール → ログイン認証。Free プランなら無料で始められます（所要時間 約 10 分）。詳細手順は公式ドキュメントへ。

ステップ 1 — Whop でアカウント登録

AlphaForge は Whop でのアカウント登録が必要です（Free プランも含めて全プラン共通）。まずは無料プランで登録し、CLI を動かしてから必要に応じて有料プランへアップグレードできます。

無料で Whop に登録 →

最初から有料プランで始めたい場合は 有料プラン一覧 から購入してください。Free / 有料の機能差は フリーミアム制限 を参照。

ステップ 2 — CLI をインストール

ターミナルで実行します。インストーラーが最新バイナリをダウンロードし、/usr/local/bin に配置します。

curl -sSL https://alforge-labs.github.io/install.sh | bash

PowerShell（管理者権限不要）で実行します。バイナリを %USERPROFILE%\.forge\bin にインストールし、PATH を自動設定します。

irm https://alforge-labs.github.io/install.ps1 | iex

インストーラを使わず手動で配置したい場合は、 GitHub Releases（最新版） から各プラットフォーム向けバイナリを直接ダウンロードできます。

ステップ 3 — Whop でログイン認証

ステップ 1 で登録した Whop アカウントで OAuth 2.0 PKCE 認証を行います。ブラウザが自動で開きます。認証情報は ~/.forge/credentials.json に保存されます。

forge system auth login

認証済みメンバーシップは以下で確認できます：

forge system auth status

結果を可視化する — alpha-visualizer（任意）

alpha-visualizer は forge が出力するバックテスト結果を Web ブラウザで可視化する OSS パッケージです（PyPI 配布・MIT ライセンス）。forge 本体に依存せず単体動作するため、必要に応じて後から追加できます。 

# uv（推奨）
uv tool install alpha-visualizer

# pip
pip install alpha-visualizer

詳しい使い方・設定は alpha-visualizer インストールガイド を参照してください。PyPI: pypi.org/project/alpha-visualizer / GitHub: alforge-labs/alpha-visualizer

主要コマンド

forge backtest run
単一銘柄に戦略を適用してバックテストを実行する。
forge backtest run SYMBOL --strategy ID
--strategy--start--json
cli-reference で詳細 →
forge optimize run
Optuna によるパラメータ最適化（TPE）。多目的最適化にも対応。
forge optimize run SYMBOL --strategy ID
--metric--trials--apply
cli-reference で詳細 →
forge optimize walk-forward
時系列をウィンドウ分割し IS 最適化 → OOS 評価で過学習耐性を測定。
forge optimize walk-forward SYMBOL --strategy ID --windows N
--windows--metric
cli-reference で詳細 →
forge data fetch
OHLCV をプロバイダーから取得し Parquet で保存。
forge data fetch SYMBOL --period 5y --interval 1d
--period--interval--watchlist
cli-reference で詳細 →
forge strategy save
戦略 JSON をレジストリに登録（任意で Journal に記録）。
forge strategy save FILE_PATH
--force
cli-reference で詳細 →
forge journal show
戦略の全履歴（スナップショット・Runs・タグ・ライブサマリ）を表示。
forge journal show STRATEGY_ID
