    信頼・安全・制限¶

    AlphaForge を購入・登録・利用する前に確認してほしい要点をまとめます。法的な正本は 免責事項、プライバシーポリシー、EULA です。このページは、それらを読みやすく整理した購入前確認用のガイドです。

    ローカル実行とデータの扱い¶

    AlphaForge CLI はローカル実行を前提にしたソフトウェアです。バックテスト設定、最適化パラメータ、戦略データ、取引履歴、ポジション情報、ローカルファイル、操作ログは、通常 Alforge Labs のサーバーへ送信されません。

    ただし、外部データプロバイダーやブローカー API をユーザー自身が設定して使う場合、その通信は各プロバイダーとの間で発生します。API キーや証券会社の認証情報の管理はユーザー自身の責任です。

    ライセンス認証で送信される情報¶

    ライセンス認証時には、認証に必要な情報だけが外部ライセンスサービスへ送信されます。具体的には、forge license activate の実行時にライセンスキー、インスタンス名、アクティベーション日時などが送信されます。

    バックテスト設定、API キー、取引履歴、戦略データはライセンス認証のために送信されません。詳細は プライバシーポリシー を参照してください。

    投資助言ではありません¶

    AlphaForge は、戦略研究・バックテスト・最適化を支援する開発者向けソフトウェアです。金融助言、売買シグナル、運用受託、証券仲介サービスではありません。

    バックテスト、最適化、シミュレーション結果は過去データまたは仮想条件に基づくものであり、将来の成果を保証しません。売買判断、銘柄選択、パラメータ設定、ポジションサイズ、執行方法はすべてユーザー自身の責任で決定してください。

    バックテストの限界¶

    バックテストでは、実市場のすべての条件を完全には再現できません。特に以下の要素は、実運用の結果に大きく影響します。

    • スリッページ、手数料、スプレッド
    • 約定遅延、流動性、部分約定
    • データ品質、欠損、タイムゾーン差
    • 過剰最適化、カーブフィッティング
    • 税務、規制、証拠金、強制決済

    検証結果をそのまま実運用の期待値として扱わず、ウォークフォワード分析、コスト設定、外部期間検証、少額検証などを組み合わせてください。

    プラン制限¶

    AlphaForge には Free / Monthly / Annual / Lifetime の 4 プランがあります。Free プランでは、データ取得・評価の日付が 2023-12-31 までに制限され、最適化 trial 数は 50 trials までに制限されます。

    Monthly / Annual / Lifetime は有料プランとして扱われ、最新データと無制限 trial を利用できます。詳細な対象コマンド、JSON 出力、開発時の確認方法は フリーミアム制限 を参照してください。
