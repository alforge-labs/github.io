# 目的別ユースケース

あなたの目的や技術的なバックグラウンドに合わせた入口を選んでください。

| ペルソナ | こんな人に向いている |
|---------|-------------------|
| [TradingViewユーザー](tradingview.md) | TradingViewでPine Scriptを書いており、バックテストを本格化させたい |
| [Python開発者](python-dev.md) | Pythonで戦略ロジックを書き、CLI/JSONベースで実験を管理したい |
| [クオンツ・研究者](quants.md) | 統計的な厳密性を重視し、最適化・ウォークフォワード検証を体系的に行いたい |
| [自動売買検討者](auto-trading.md) | バックテストから実運用（自動発注）まで一貫した環境を構築したい |
| [AIエージェント利用者](ai-agents.md) | Claude Codeなどと組み合わせて戦略探索を自動化したい |
| [裁量トレーダーのシステム化](discretionary.md) | 頭の中の裁量ルールを JSON で明示化し、感情バイアスを定量検証したい |
| [他ツールとの比較](comparison.md) | Backtrader・vectorbt・TradingViewとの使い分けを知りたい |
