# Use Cases by Goal

Choose the entry point that matches your background and goals.

| Persona | Best for |
|---------|---------|
| [TradingView Users](tradingview.md) | You write Pine Scripts in TradingView and want to upgrade your backtesting |
| [Python Developers](python-dev.md) | You write strategy logic in Python and want CLI/JSON-based experiment management |
| [Quants & Researchers](quants.md) | You prioritize statistical rigor and want systematic optimization and walk-forward validation |
| [Automated Trading](auto-trading.md) | You want an end-to-end environment from backtesting to live order execution |
| [AI Agent Users](ai-agents.md) | You want to automate strategy exploration with Claude Code or similar AI agents |

---

For market-specific use cases (Futures, Equities, FX), see the **Activation Scenes** section on the [homepage](https://alforge-labs.github.io/en/).
